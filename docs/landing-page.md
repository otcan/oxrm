# Landing Page Copy

Use this as the minimal shared launch page content for Orkestr plus oXRM.

## Headline

Local-first software for running agents against real workflows.

## Subheadline

Orkestr gives agents a workstation. oXRM gives them relationship memory.

## Orkestr

Local-first agent workstation with persistent Codex threads, secure remote
access, browser control, WhatsApp routing, and timers.

Quickstart:

```bash
git clone https://github.com/otcan/orkestr.git
cd orkestr
./scripts/install.sh --local
```

## oXRM

Self-hosted outreach workspace for job search, customer outreach, partnerships,
and founder-led sales.

Quickstart:

```bash
git clone https://github.com/otcan/oxrm.git
cd oxrm
./oxrm start
./oxrm ready
./oxrm seed job-search
./oxrm test
./oxrm urls
```

## Safety Note

Both projects are self-hosted and public-demo-safe by default. Public examples
must use synthetic data only. Do not expose raw local services directly to the
public internet.
